# 퉁퉁퉁퉁! 소리의 과학 게임 실험실 🎵

> 오늘의 과학자가 내일의 과학자를 만난다!

초등학생을 위한 소리와 리듬을 배우는 교육용 게임입니다. 진동, 파동, 주파수의 원리를 재미있게 체험할 수 있습니다.

## 🌟 라이브 데모

**👉 [https://oct-sky2025.vercel.app/](https://oct-sky2025.vercel.app/)**

## 🎮 게임 모드

### 1. 소리의 과학 배우기 🎓
- **소리란 무엇일까?** - 진동과 파동의 기본 원리
- **진동을 느껴보세요** - 실생활 체험 활동
- **주파수와 음의 높낮이** - Hz의 개념 이해
- **리듬이란?** - BPM과 박자 학습

### 2. 게임 설정하기 ⚙️
**난이도 프리셋**
- 🟢 **초급**: BPM 80, 쉬운 판정
- 🟡 **중급**: BPM 110, 보통 판정
- 🔴 **고급**: BPM 140, 어려운 판정

**사운드팩 선택**
- 🥁 드럼
- 🎹 피아노
- 🚀 우주선
- 🐾 동물

**테마 선택**
- ☁️ 하늘
- 🌌 우주
- 🌊 바다
- 🌲 숲

**기타 옵션**
- 📳 진동 피드백
- 👁️ 가이드선 표시

### 3. 리듬 게임 시작 🎮
**게임 방법**
1. 화면 중앙의 동그라미를 주목하세요
2. 빛의 고리가 점점 작아지며 중앙으로 모입니다
3. 고리가 중앙 원에 딱 맞을 때 하단 버튼을 탭!
4. 정확도에 따라 점수를 받습니다

**판정 시스템**
- 💖 **Perfect**: 완벽한 타이밍! (100점)
- 💙 **Great**: 훌륭해요! (70점)
- 💚 **Good**: 좋아요! (50점)
- 🩶 **Miss**: 다시 도전! (0점)

**점수 계산**
- 기본 점수 + 콤보 보너스 (최대 50점)
- 콤보는 연속 성공 시 증가
- Miss 시 콤보 초기화

### 4. 소리 맞히기 게임 🎯
**게임 방법**
1. 🔊 스피커 아이콘을 클릭하여 소리를 들어보세요
2. 4개의 선택지 중 정답을 고르세요
3. 총 3문제로 구성되어 있습니다

**문제 유형**
- 빗소리 (쪼로롱 쪼로롱)
- 강아지 짖는 소리 (멍멍!)
- 드럼 소리 (둥! 둥!)

### 5. 가족 대항전 🏆
**팀 대결 모드**
- 최대 3팀이 동시에 경쟁
- 각 팀의 점수와 최대 콤보 기록
- 실시간 순위 업데이트
- 세션 코드로 여러 기기 연동 가능

**규칙**
- ✅ 공정하게: 모든 팀은 같은 난이도로 시작
- ✅ 안전하게: 차례를 지켜주세요
- ✅ 응원하며: 함께 응원해요!

## 🎵 소리 시스템 특징

- **Web Audio API** 기반 실시간 사운드 생성
- 메트로놈 비트는 조용하게 (볼륨 0.1)
- 판정 소리는 명확하게 (볼륨 0.25)
- 진동 피드백으로 촉각 경험 제공

## 🛠️ 기술 스택

- **Frontend**: React 18 (CDN)
- **Styling**: Tailwind CSS
- **Database**: Supabase (실시간 동기화)
- **Audio**: Web Audio API
- **Deployment**: Vercel
